Travel Bucket List

Technical project built using Spring Tool Suite and Java with MySQL as a database. Started by building a wireframe and features plan. Features include adding items to your list, browsing items, login/registration, and sorting your table of items.

Trekventure

Built a single page text adventure game using session and MySQL database with the ability to login, save a game, and reload. Developed a login system via MySQL database, RegEx, validations and bcryption. Utilized STAPI API to generate space objects.

https://github.com/Jrrohr/Trekventure
